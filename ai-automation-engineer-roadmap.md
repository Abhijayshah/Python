# 🎯 Top 20 Interview Questions for AI/Automation Engineers

| # | Question | Short Answer |
| :--- | :--- | :--- |
| 1 | *"What is the difference between `requests` and `selenium`?"* | `requests` = HTTP only, `selenium` = controls actual browser (renders JS) |
| 2 | *"How do you handle a pop-up or CAPTCHA?"* | CAPTCHA needs 3rd party service (2Captcha). Pop-ups: `driver.switch_to.alert` |
| 3 | *"What is the GIL? Does it affect AI training?"* | GIL blocks true multi-threading. For AI, we use multiprocessing or GPU (not affected) |
| 4 | *"Explain overfitting. How do you prevent it?"* | Model memorizes training data. Prevent with: more data, cross-validation, regularization, dropout |
| 5 | *"What is a confusion matrix?"* | Table showing True Positives, False Positives, True Negatives, False Negatives |
| 6 | *"What is the difference between `fit()` and `transform()`?"* | `fit()` learns parameters, `transform()` applies them |
| 7 | *"How do you handle imbalanced dataset?"* | SMOTE, class weights, oversampling minority class, undersampling majority |
| 8 | *"What is an API? How do you call one in Python?"* | API = Application Programming Interface. Call with `requests.get(url)` |
| 9 | *"How do you schedule a Python script?"* | `schedule` library, `cron` (Linux), Task Scheduler (Windows), `APScheduler` |
| 10 | *"What is headless browser? When to use?"* | Browser without UI. Use on servers for faster scraping |
| 11 | *"Explain Precision vs Recall."* | Precision = "of predicted positives, how many correct?" Recall = "of actual positives, how many found?" |
| 12 | *"What is RAG (Retrieval Augmented Generation)?"* | Give LLM relevant context from your documents before asking question |
| 13 | *"How do you avoid getting blocked while web scraping?"* | Rotate user-agents, use delays (`time.sleep`), proxies, respect `robots.txt` |
| 14 | *"What is the difference between `@app.route` and a decorator?"* | `@app.route` IS a decorator. It wraps a function with URL mapping logic |
| 15 | *"How do you save and load a trained model?"* | `pickle.dump(model, file)` or `joblib.dump()` |
| 16 | *"What is a tensor?"* | Multi-dimensional array (like NumPy but for GPU). Core data structure in TensorFlow/PyTorch |
| 17 | *"What is the difference between CNN and RNN?"* | CNN = spatial data (images), RNN = sequential data (time series, text) |
| 18 | *"How do you handle dynamic content in web scraping?"* | Use Selenium/Playwright, or find the hidden API that loads the data |
| 19 | *"What is the OpenAI API rate limit? How to handle?"* | Use `time.sleep()`, implement retry logic with exponential backoff |
| 20 | *"Why Docker for AI/Automation?"* | Reproducible environment. "It works on my machine" problem solved |
